FAQ

Preguntas frecuentes

¿Necesito contrato propio con MRW / Correos / DHL? +
Sí. Enviox conecta con TU contrato existente con el transportista. No somos intermediarios — usamos la API de tu cuenta para generar etiquetas y gestionar envíos a tus precios negociados. Necesitas tener credenciales API activas.
¿Funciona con Shopify, WooCommerce y PrestaShop? +
Sí, tenemos apps y plugins para las 3 plataformas principales en España. Cada producto está optimizado para su plataforma: Shopify Polaris para Shopify, WordPress admin para WooCommerce, y PrestaShop back-office para PrestaShop.
¿Qué es el Recargo de Equivalencia? +
El RE es un régimen especial de IVA obligatorio en España para comerciantes minoristas. Los tipos actuales son 5.2% (general), 1.4% (reducido) y 0.5% (superreducido). Nuestras apps lo calculan y facturan automáticamente.
¿Puedo usar varios transportistas a la vez? +
¡Sí! Puedes instalar MRW Pro, Correos Pro y DHL Pro simultáneamente. Cada app funciona independientemente. Con el plan Business puedes usar el motor de reglas para enrutar pedidos al transportista más adecuado automáticamente.
¿Cuánto tarda la instalación? +
Menos de 2 minutos. Instala la app, introduce tus credenciales API del transportista, configura las reglas básicas y empieza a enviar. Sin configuraciones complicadas.
¿Ofrecéis soporte en español? +
Por supuesto. Somos un equipo 100% español. Soporte por email en todos los planes, y soporte prioritario en planes Profesional y Business. Respondemos en menos de 24h (normalmente en minutos).
