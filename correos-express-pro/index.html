FAQ

Preguntas sobre Correos Express Pro

¿Necesito un contrato con Correos Express?+
Sí. Correos Express Pro conecta con tu contrato existente con Correos Express. Necesitas tener credenciales API activas (usuario, contraseña y código de cliente). Si no las tienes, contacta a tu comercial de Correos Express.
¿Qué diferencia hay con Correos Pro?+
Correos Pro integra Correos Standard (el servicio postal tradicional, más económico). Correos Express Pro integra Correos Express (la filial de paquetería urgente, con entrega 24h). Son empresas distintas con APIs diferentes. Si necesitas ambos, puedes instalar las dos apps y usar el motor de reglas para decidir cuál usar.
¿Qué servicios de Correos Express soporta?+
Soportamos: Paq Premium (24h), Paq Empresa 14 (antes de las 14h), ePaq 24 (ecommerce 24h), Paq Estándar (48-72h), International Express y gestión de devoluciones.
¿Puedo usar Correos Express Pro junto con MRW Pro?+
¡Sí! Puedes instalar MRW Pro, Correos Pro, Correos Express Pro y DHL Pro simultáneamente. Cada app funciona de forma independiente. Con el ecosistema Enviox, puedes usar el motor de reglas para enrutar al carrier óptimo automáticamente.
¿Cuánto tarda la instalación?+
Menos de 2 minutos. Instala la app → introduce tus credenciales Correos Express → selecciona el servicio por defecto → listo. Tu primer envío puede estar creado en los primeros 5 minutos.
