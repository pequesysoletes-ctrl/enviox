¿Dónde encuentro mis credenciales?

Sigue los pasos según tu tipo de contrato con Correos Pro para obtener acceso a la API.

Correos Standard

public

Paso 1

Accede a empresas.correos.es > Mi cuenta

settings

Paso 2

Ve a Integraciones > API Ecommerce

key

Paso 3

Tus credenciales están en el apartado de Pre-registro API

Correos Express

public

Paso 1

Accede a correosexpress.es > Área privada

settings

Paso 2

Ve a Integraciones

key

Paso 3

Solicita las credenciales API a tu delegación Correos Express

info
Si tienes contrato activo con Correos o Correos Express, las credenciales existen. Si ya usaste la app oficial de Correos, tus credenciales son las mismas. Contacta con tu gestor comercial si necesitas ayuda.
