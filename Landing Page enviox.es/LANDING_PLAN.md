> **Dominio:** enviox.es

> **Estado:** 📋 Planificación  
> **Fecha:** 19/02/2026

---

## 1. Objetivo de la Landing

La landing page de enviox.es tiene 3 funciones:

1. **Escaparate de productos** — Mostrar todos los productos Enviox con links a cada App Store/marketplace
2. **SEO** — Capturar tráfico orgánico de búsquedas como "app envíos Shopify España", "MRW Shopify", etc.
3. **Confianza** — Cuando un comerciante ve la app en el App Store y visita la web, debe sentir que es una empresa seria

## 5. Stack Técnico

| Aspecto | Decisión | Razón |
|---------|----------|-------|
| **Framework** | HTML/CSS/JS estático | Velocidad, SEO, sin server needed |
| **Alternativa** | Astro o Next.js (SSG) | Si necesitamos blog dinámico |
| **Hosting** | VPS existente (Nginx) | Coste 0€ |
| **SSL** | Let's Encrypt (certbot) | Gratis |
| **Formularios** | Formspree o email API | Sin backend |
| **Analytics** | Plausible o GA4 | Métricas tráfico |
| **Blog** | Markdown → HTML (o Astro) | SEO articles |

### DNS (cuando el dominio esté activo)

```
enviox.es          A       → IP del VPS
www.enviox.es      CNAME   → enviox.es
app.enviox.es      A       → IP del VPS (backend apps)
docs.enviox.es     CNAME   → (futuro)
```

- Moderno, limpio, profesional
